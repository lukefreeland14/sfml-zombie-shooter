# 2D Zombie Shooter (C++ / SFML)

A 2D zombie shooter built in C++ using SFML.

## Features
- Object-oriented architecture
- Enemy wave progression system
- Collision detection
- Projectile physics
- Sprite-based animation

## Technologies
- C++
- SFML
- Object-Oriented Programming
